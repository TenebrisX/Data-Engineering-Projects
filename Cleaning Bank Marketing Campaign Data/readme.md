# Bank Marketing Data Preparation

This project cleans and prepares data from a bank's marketing campaign, making it ready for import to a relational database.

## Project Overview

Personal loans are a significant source of revenue for banks. This project focuses on cleaning raw data collected during a bank's personal loan marketing campaign. The goal is to prepare the data in a format suitable for creating a well-structured database, facilitating future marketing campaigns and analysis.

## Data Source

The data for this project is contained in a CSV file named `bank_marketing.csv` From [DataCamp](https://app.datacamp.com/learn/projects/1613)

## Final Data Structure

The cleaned data is split into three CSV files:

*  **client.csv**
*  **campaign.csv**
*  **economics.csv**
